# 3D Animation Engine

Собственный расширяемый 3D-движок для визуализации анимаций в реальном времени, написанный на C++ с использованием OpenGL.

## Идея
Проект представляет собой каркас (framework) для создания 3D-сцен. Архитектура построена по принципу модульности, где каждый элемент сцены (Unit) является независимой сущностью.

### Технические особенности:
- **Архитектурные паттерны**: 
    - **Singleton**: центральный класс движка `anim` доступен из любой точки программы.
    - **Registry Pattern**: автоматическая регистрация объектов (Units) в системе через статические конструкторы.
- **Шаблонная математика**: собственная библиотека линейной алгебры в папке `mth/` (векторы, матрицы, работа с камерой).
- **Управление ресурсами**: обобщенный менеджер ресурсов для шейдеров, текстур, моделей и материалов.
- **Оптимизация данных**: использование **UBO (Uniform Buffer Objects)** для эффективной передачи данных камеры и материалов в шейдеры.
- **Системный уровень**: прямая работа с **WinAPI** для создания окна, обработки системных сообщений и высокоточного замера времени.

## Структура проекта
```
T06ANIM/
├── src/
│   ├── anim/                   # Ядро системы анимации
│   │   ├── input/              # Обработка ввода (Keyboard, Mouse) и Timer
│   │   ├── rnd/                # Графический пайплайн (Render)
│   │   │   └── res/            # Менеджеры ресурсов (Реализация CRUD для GPU)
│   │   │       ├── buf/        # Управление UBO (Uniform Buffer Objects)
│   │   │       ├── mtl/        # Система материалов
│   │   │       ├── prim/       # Примитивы, модели и работа с топологией
│   │   │       ├── shd/        # Загрузка и компиляция шейдеров (GLSL)
│   │   │       └── tex/        # Загрузка изображений и создание текстур
│   │   ├── anim.cpp / .h       # Главный класс движка (Singleton, Registry)
│   │   └── resource.h          # Базовый шаблонный класс для всех менеджеров
│   ├── mth/                    # Header-only библиотека линейной алгебры
│   │   ├── mth_matr.h          # Матричные преобразования
│   │   ├── mth_cam.h           # Математическая модель камеры
│   │   └── mth_vec*.h          # Векторная математика (vec2, vec3, vec4)
│   ├── units/                  # Пользовательские объекты сцены (Юниты)
│   ├── win/                    # Низкоуровневая обертка над WinAPI
│   ├── def.h                   # Глобальные определения и POD-структуры для UBO
│   └── main.cpp                # Точка входа (WinMain)
├── T06ANIM.sln                 # Решение Visual Studio
├── bin/                        # Ресурсы (модели, текстуры, шейдеры)
└── README.md
```

## Технологический стек
- **Язык**: C++17.
- **Графическое API**: OpenGL 4.6 (GLSL).
- **Платформа**: Windows (WinAPI).
- **Сборка**: Visual Studio.

## Как запустить
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/VG6VG6/SUMCAMP2024.git
   ```
2. Откройте файл `T06ANIM.sln` в Visual Studio.
3. Убедитесь, что все зависимости (OpenGL, GLEW) настроены.
4. Соберите и запустите проект (рекомендуется конфигурация **Release**).
